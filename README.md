Здесь я делюсь своими скриптами для psexec и не только, которые немного облегчают мне жизнь при работе в технической поддержке.

Скрипты по большей части нацелены на то, чтобы взаимодействовать с доменными ПК без подключения к ним по RDP.

### **IP Scanner**
Отделом ИБ были заблокированы полноценные сканеры IP диапазонов, поэтому пришлось написать своё простенькое решение.

Выводит IP - Hostname - Описание посленего залогинившегося пользователя.

### **Расшаривание принтера**
Выводит список подключенных к удалённому ПК принтеров, и позволяет расшарить необходимый.

### **Сигнал о появлении в сети**
Выводит поверх всех окон сообщение, когда нужный ПК появится в сети.
